Taksimetre Tutarını Hesaplama

Proje Açıklaması
Bu proje, bir taksimetre sisteminin ücret hesaplama işlevini gerçekleştiren basit bir Java uygulamasıdır. Uygulama, kullanıcıdan mesafeyi alarak toplam taksimetre ücretini hesaplar. Hesaplanan ücret, belirli bir minimum ücret ile karşılaştırılarak en az bu miktar olacak şekilde ayarlanır.

Özellikler

Mesafe Girişi: Kullanıcıdan mesafeyi kilometre cinsinden alır.

Ücret Hesaplama: Mesafeye bağlı olarak toplam ücret hesaplanır.

Minimum Ücret: Hesaplanan ücret belirli bir minimum ücretin altındaysa, minimum ücreti kullanır.

Gereksinimler

Java Development Kit (JDK): Projeyi derlemek ve çalıştırmak için Java JDK 8 veya üstü gereklidir.

Kurulum ve Çalıştırma

Kodun Yüklenmesi:

Kodunuzu bir Java IDE (IntelliJ IDEA, Eclipse vb.) ya da bir metin düzenleyiciye yapıştırın ve dosyayı Main.java olarak kaydedin.

Derleme:

Komut satırını açın ve dosyanın bulunduğu dizine gidin.

Java dosyasını derlemek için şu komutu çalıştırın:

javac Main.java

Çalıştırma:

Derlenmiş sınıf dosyasını çalıştırmak için şu komutu kullanın:

java Main

Kullanım

Mesafe Girişi:

Program çalıştırıldığında kullanıcıdan mesafeyi kilometre cinsinden girmesi istenir.

Ücret Hesaplama:

Kullanıcı tarafından girilen mesafeye göre toplam ücret hesaplanır.

Hesaplanan ücret, belirlenen minimum ücret olan 20 TL ile karşılaştırılır. Eğer hesaplanan ücret 20 TL'den düşükse, 20 TL olarak ayarlanır.

Sonuç:

Hesaplanan veya ayarlanan toplam ücret ekranda iki ondalıklı formatta gösterilir.
